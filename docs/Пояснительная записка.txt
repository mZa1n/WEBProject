Наш проект поделился на 2 сектора:
1) Телеграм бот
2) Сайт на микрофреймворке flask

Телеграм бот основан на библиотеке python-telegram-bot и из функций в нем есть:
1)Функция start (отвечает за приветствие пользователя и небольшой подсказки
2)Функция help (список команд)
3)Функция linking_to_a_bot (позволяет пользователю привязать свой аккаунт к боту в тг)
4)Функция check_tasks (выводит самую близжайшую по сроку задачу)
5)Функция error (выводит ошибки)

Сайт состоит из 14 основных функций:
1)Функция main_page (отвечает за главную страницу с новостями сайта)
2)Функция index (отвечает за вывод задач пользователя)
3)Функция profile (отвечает за отрисовку профиля пользователя)
4)Функция give_admin (отвечает за выдачу админ-прав пользователю)
5)Функция register (отвечает за регистрацию на сайте)
6)Функция load_user (отвечает за прогрузку пользователя)
7)Функция login (отвечает за вход на сайте)
8)Функция add_tasks (отвечает за добавление задачи)
9)Функция news (отвечает за добавление новости на сайте)
10)Функция delete_news (отвечает за удаление новости)
11)Функция tasks_delete (отвечает за удаление задачи)
12)Функция link (отвечает за прогрузку страницы с информацией для подключения бота)
13)Функция logout (отвечает за выход пользователя из профиля)
14)Функция generate_code (генерирует токен для добавления бота)
